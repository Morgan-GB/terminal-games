# Terminal Games

A few simple games that can be played in the terminal!

# Games

- [ ] Tic Tac Toe
- [ ] Connect 4

# How to Run

1. Firstly, clone this github repository:

```bash
git clone https://github.com/MorganWJones/terminal-games
```

2. Build the games:

```bash
# NPM
npm run build

# Yarn
yarn build

# Rollup
rollup -c ./rollup.config.js
```

3. Get playing!:

```bash
# List all available games
node . --list

# Play a game
node . [game-name]
```
